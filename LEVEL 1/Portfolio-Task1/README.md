
# 💼 Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML and CSS** to showcase skills, projects, and contact information.

This portfolio represents my journey as a **Frontend Developer**, highlighting my technical skills and project work.

---

## 🌟 Features

* ✅ Responsive Navigation Bar
* ✅ Hero Section with Introduction
* ✅ About Me Section
* ✅ Skills Showcase Grid
* ✅ Projects Gallery
* ✅ Contact Section
* ✅ Resume Download Button
* ✅ Fully Responsive Design

---

## 🛠 Technologies Used

* **HTML5** – Structure and semantic layout
* **CSS3** – Styling, layout, and responsiveness
* **Unsplash Images** – Project thumbnails

---

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Make sure all files are in the same folder.
3. Open `index.html` in your browser.
4. Explore the portfolio sections.
