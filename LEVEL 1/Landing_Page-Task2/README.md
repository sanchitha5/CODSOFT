
# 🌐 Modern Landing Page

A clean and responsive **Modern Landing Page** built using **HTML and CSS**.
This project demonstrates a structured layout with a navigation bar, hero section, features section, call-to-action area, and footer.

---

## 📌 Features

*  Responsive Navigation Bar
*  Attractive Hero Section with Image
*  Features Section with Cards
*  Call-To-Action (CTA) Section
*  Clean Footer
*  Fully Responsive Layout
*  Modern UI Design

---

## 🛠 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and responsiveness
* **Unsplash Image** – Hero section image

---

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Ensure `index.html` and `style.css` are in the same folder.
3. Open `index.html` in your browser.
4. View the landing page in desktop or mobile view.
